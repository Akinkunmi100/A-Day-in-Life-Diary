# Telecom Ethnography: Thematic Analysis Project

## Project Overview
This repository contains the full thematic analysis pipeline for a Lagos-based telecom ethnographic study. The study focuses on 65 diary entries from 11 respondents, analyzed using a 6-phase reflexive thematic analysis framework (Braun & Clarke).

## Key Deliverables
- **Unified Codebook**: 11-family (A-K) thematic system defining work, productivity, digital finance, power infrastructure, and social behavior.
- **Respondent Personas**: 11 high-rigour, narrative-driven persona reports (`.docx`) including archetypes like "The Hustling Provider" and "The Tired Taskmaster."
- **Coded Corpus**: Full segment-level coding for all 65 entries (`coded_corpus_full.xlsx`).
- **Visual Analytics**: Journey maps and word clouds for each respondent.

## Repository Structure
- `scripts/`: Python engine for data extraction, automated coding, and persona generation.
- `unified_codebook.md`: Master reference for all thematic definitions.
- `*.docx`: Final persona reports and analysis guides.
- `*.png`: Visualizations and journey maps.

## Automation Pipeline
The project utilizes a custom Python-based coding engine that:
1. Extracts diary entries from Excel.
2. Applies the 11-family coding system at the segment level.
3. Generates structured persona reports in Word format with custom styling.
4. Produces journey maps and frequency-based word clouds.
